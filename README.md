# Laterr

## Routes

The application routes are defined in `src/App.tsx` and map as follows:

| Path | Component |
| --- | --- |
| `/` | `Dashboard` |
| `/dashboard` | `Dashboard` |
| `/app` | `Bookmarks` |
| `/landing` | `Landing` |
| `/auth` | `Auth` |
| `/bookmarks` | `Bookmarks` |
| `/subscriptions` | `Subscriptions` |
| `/health` | `Health` |
| `/settings` | `Settings` |
| `*` | `NotFound` |
